# Growthiva Webpage

Este repositorio contiene el código fuente de la página web de Growthiva. Puedes visitar la página en el siguiente enlace: [Growthiva Webpage](https://growthiva-webpage.vercel.app).

## Características
- **Página web estática:** Construida con HTML y CSS.
- **Diseño limpio y funcional.**

## Cómo usar
1. Visita el sitio web directamente en [https://growthiva-webpage.vercel.app](https://growthiva-webpage.vercel.app).
2. Si deseas modificar o desarrollar la página localmente, clona el repositorio:
   ```bash
   git clone https://github.com/juangpazx/growthiva-webpage.git
   ```
